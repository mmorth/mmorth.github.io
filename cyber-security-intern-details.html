Cyber Security Intern

John Deere
Summer 2018

During the summer of 2018, I worked as a Cyber Security Intern on the security architecture team in Moline, Illinois. Below is a list and description of the major projects I worked on and skills that I learned during this internship:

  • AWS WorkSpace Security Group Automation: Completed
  • Setup and Integrate Cloud Security Tool: Completed
  • IoT Cryptography Method Research: Completed

AWS WorkSpace Security Group Automation

  • Description: The first project I worked on was automating the process of attaching necessary security groups to newly provisioned AWS WorkSpaces.
  • Results: I was able to complete the automatic assigning of security groups when a WorkSpace is created. After my internship, my work would be integrated into the create workspace internal web form.
  • Impact: This project helped speed up the process of creating AWS WorkSpaces and increased security by only allowing access to required services.

Setup and Integrate Cloud Security Tool

  • Description: The second project I worked on was configuring and integrating a cloud security tool into the company's network.
  • Results: I successfully configured and integrated the cloud security tool into the company's network.
  • Impact: The cloud security tool will help monitor network traffic to help detect and prevent network attacks and vulnerabilities.

IoT Cryptography Method Research

  • Description: The third project I worked on was to compare two cryptography methods for use in IoT devices.
  • Results: I successfully research, documented, and presented my research and recommendations to the cloud security team.
  • Impact: This research and recommendations helped the cloud security team to select a cryptography method to use in various IoT devices.
