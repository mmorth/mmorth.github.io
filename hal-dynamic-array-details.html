Horn and Lights (HAL)/Dynamic Array

PrISUm Solar Car
Fall 2016 - Spring 2017

Horn and Lights (HAL) was the first project I worked on while at PrISUm solar car. As the name suggests, the project controls the horn and lights of the vehicle. I worked on the code related to this project. The Dynamic Array project controls the orientation of the solar array cells to achieve optimal orientation with the sun. I worked on the code that determines the direction of the solar arrays.

Overview

  • Description: HAL controls the horn and lights of the car. Dynamic Array controls the orientation of the solar arrays toward the sun.
  • Results: I was able to complete and integrate HAL into the final design and was able to create the foundation code for the dynamic array.
  • Impact: The HAL project allowed operators to control the horn and lights, allowing the vehicle to be street legal.
