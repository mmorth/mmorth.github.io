Basketball Simulation

Personal Project
Summer 2018

Basketball Simulation is a web application that allows the user to create teams, players, and coaches that allows the simulation of games between teams. The simulation tracks scores by possession, quarter, and game while also tracking player's points, rebounds, assists, and blocks during the game.

Overview

  • Description: Basketball Simulation is a web application that allows the user to create players and teams and simulate games between them to see who would win and player/team stats.
  • Results: I completed the player and team creation, team simulation, and team and player stat tracking.
  • Impact: This is a for fun application that I use for entertainment from time to time.


    • Features

    • Player, Coach, and Team creation
    • Simulation between two created teams
    • Player and team stat tracking
    • Improved simulation algorithm with player fatigue and substitutions
