Basketball Dribble Coach

Personal Project
Spring 2021

Basketball Dribble Coach is a C++ OpenCV Computer Vision Qt application that will track the postion of the player's body, face, and the basketball. The application will display the number of completed repetitions, number of times the player has looked down while dribbling, and the elasped time of the drill.

Overview

  • Description: Basketball Dribble Coach is a C++ OpenCV Computer Vision Qt application that will track the postion of the player's body, face, and the basketball. The application will display the number of completed repetitions, number of times the player has looked down while dribbling, and the elasped time of the drill.
  • Results: I was able to create an application that correctly counts over 90% of the repetitions and over 70% of the number of times the player looks down.
  • Impact: This project is a fun project that myself and other basketball players can use to improve their basketball dribbling skills.


    • Features

    • Basketball position tracking using color tracking
    • Player position tracking using background subtraction
    • Look down detection using Haar Cascade Classifiers
    • Object intersection detection to count the number of completed repetitions
    • Support for different environments, lighting conditions, and clothing
