CyStudy

College Project
Spring 2018

CyStudy is a study group finder web application that I worked on for a software project class that I completed at Iowa State University. I managed the back-end and database of the system. The web app allows the user to create, find, and join study groups for classes in which they are enrolled.

  • GitHub:Â https://github.com/mmorth/CyStudy

Overview

  • Description: CyStudy is a web application that allows a user to create, find, and join study groups for courses they are enrolled in. Once they join a group, they can create meetings, chat in real-time, and view notes.
  • Results: Our group was able to complete all the major features of the project.
  • Impact: This web app was not used by other students, but it did receive an honorable mention for the best project.


    • Features

    • User authentication using Django authentication
    • Study group finding, creation, and joining
    • Meeting creation, real-time messaging, note creation, and building identifier within a study group
    • Course admin and moderator functionality to manage study groups and members
